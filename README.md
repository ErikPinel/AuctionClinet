## Running the project

1. Clone the repo.
2. Run `npm install`.
3. enter the server folder.
4. run `nodemon server.js`.
5. go back to the root folder.
6. Run `npm start`.
7. Navigate to `http://localhost:3000`.

## Technologies:

### Client-side:
* React.JS
* mui
* nivo charts

### Server-side:
* Node.JS
* Express
* MongoDb
